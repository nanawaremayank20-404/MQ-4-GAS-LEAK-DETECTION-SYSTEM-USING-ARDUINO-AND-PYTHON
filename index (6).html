🔥 Gas Leak Detection System

Arduino Uno + MQ-4 + Python Dashboard

About

This project detects methane gas using an MQ-4 sensor connected to an Arduino Uno. Readings are sent to a Python dashboard for real-time monitoring and alerts.

Features

  • Real-time Monitoring
  • Python Dashboard
  • Gas Leak Alerts
  • CSV Logging
  • Serial Communication

Hardware

  • Arduino Uno
  • MQ-4 Sensor
  • Breadboard
  • Jumper Wires
